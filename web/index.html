SIAGC

Sistema de información para la administración Ganeadera de Cómbita

Bienvenidos!!

                    Este es un sistema para la gestión y administración Bovina dispuesto para 
                    el municipio de Cómbita en Boyacá, el sistema permite el registro a funcionarios 
                    del sistema los cuales pueden tener acceso a la mayor parte de la información 
                    permitiendo la gestión y acción de actividades en pro de la producción Bovina en 
                    la región, tambien permite el acceso a usuarios que tengan bovinos o dueños de 
                    fincas y hatos con dichos animales los cuales ingresarán los datos de los 
                    semovientes, teniendo asi valances de información disponibles para la toma de 
                    desiciones en sus terrenos.
                  
                    A continuacion encontrará un panel con tres botones en donde:

                        1.  iniciar sesión: le permite acceder las opciones principales bajo una 
                                            cuenta de usuario que usted ha creado.
                        2.  registrarme:  le permite crear una cuenta de usuario, si aun no la tiene.
                                          dicha cuenta puede ser como funcionario del sistema o como 
                                          dueño de finca o hato.
                        3.  Descargar APK Movíl:  le permite descargar la aplicacion movíl la cual 
                                                  le permite registrar las cabezas de ganado con las 
                                                  que cuenta y asi mismo ingresar los datos de 
                                                  actividades relacionadas a estos semovientes.

                    si desea conocer mas acerca del manejo de este sistema  puede dirigirse al link Información
